# Design Philosophy

The Bliss Glow visual identity must communicate:

- Premium quality
- Elegance
- Softness
- Trust
- Modern aesthetics
- Luxury without excess
- Simplicity
- Emotional connection

Every interface element must feel intentional and refined.

The design should prioritize whitespace, readability, accessibility, and conversion.

The visual language should be timeless rather than trend-driven.

---

# Typography

The typography must feel elegant and highly readable.

Recommended hierarchy:

Heading 1

Large, bold, premium appearance.

Heading 2

Strong section titles.

Heading 3

Component titles.

Body

Comfortable reading size.

Caption

Small descriptive text.

Button Text

Bold and highly legible.

Navigation Text

Medium weight with generous spacing.

---

# Border Radius

Buttons

Medium rounded corners.

Cards

Soft rounded corners.

Input Fields

Rounded corners consistent with buttons.

Images

Slightly rounded to maintain elegance.

---

# Shadows

Shadows must be subtle.

Avoid heavy shadows.

Use soft elevation to create depth.

Hover states may slightly increase elevation.

---

# Spacing System

Use a consistent spacing scale.

Small spacing

Medium spacing

Large spacing

Extra large spacing

Whitespace is essential.

Never overcrowd the interface.
