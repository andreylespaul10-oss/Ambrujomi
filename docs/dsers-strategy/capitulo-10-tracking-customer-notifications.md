# PURPOSE

Keep customers informed automatically from payment to
doorstep.

------------------------------------------------------------

# NOTIFICATION SEQUENCE

Order confirmation — instant

Shipped + tracking link — when DSers syncs the number

Out for delivery / delivered — via tracking page

"How was everything?" review request — 3 days after delivery

------------------------------------------------------------

# TRACKING RULES

Tracking numbers must be valid on a UK-friendly tracker
(17track / AfterShip style page)

If tracking shows no movement for 7 days -> proactive
customer email + supplier chase, before the customer asks

"Where is my order?" chat answers read live tracking data
