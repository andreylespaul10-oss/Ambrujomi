# PURPOSE

Import products into the store cleanly and keep supplier
links maintainable.

------------------------------------------------------------

# IMPORT WORKFLOW

Find listing -> push to DSers import list

Edit BEFORE pushing to store:
• Title rewritten in British English (no supplier keywords)
• Description rewritten to Bliss Glow tone
• Variants renamed ("Colour: Blush Pink", not "Color 2")
• Remove supplier images with foreign text or watermarks
• Set Bliss Glow pricing (see Chapter 6)

Push to store as hidden -> final visual check -> publish

------------------------------------------------------------

# MAPPING RULES

Every store product maps to a primary supplier AND a backup
supplier where possible (DSers Advanced Mapping).

Variant mapping is verified one by one before publishing.

When a supplier changes a listing, remap immediately —
never leave an unmapped variant on a live product.
