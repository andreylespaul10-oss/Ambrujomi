# PURPOSE

Explain the integration architecture so every team member
understands the order flow.

------------------------------------------------------------

# ARCHITECTURE

Bliss Glow store (Wix)

-> DSers app (installed on the store)

-> AliExpress supplier account (linked to DSers)

-> Supplier warehouse ships to customer

-> Tracking number flows back: AliExpress -> DSers -> store
-> customer notification email

------------------------------------------------------------

# KEY DSERS FEATURES USED

Product import from AliExpress with variant mapping

Supplier Optimizer — find better suppliers for the same item

Bulk order placement (pay many orders in one batch)

Automatic tracking sync

Mapping — one store product linked to multiple suppliers

Stock and price sync alerts

------------------------------------------------------------

# ACCOUNT SETUP CHECKLIST

DSers linked to the store (done)

AliExpress account linked to DSers and verified

Payment card saved in AliExpress for bulk payments

DSers notification email set to the operations inbox
