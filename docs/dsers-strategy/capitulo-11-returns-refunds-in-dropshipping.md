# RETURNS & REFUNDS IN DROPSHIPPING
# Bliss Glow DSers & AliExpress Dropshipping Strategy
# Version 1.0
# ============================================================

# PURPOSE

Handle returns premium-grade even though stock never
touches Bliss Glow.

------------------------------------------------------------

# CUSTOMER POLICY

30-day returns, no questions asked (as published on the store)

------------------------------------------------------------

# INTERNAL PLAYBOOK

Low-value item (under 15 GBP landed): refund or replace
without return — return shipping costs more than the item

Damaged/wrong item: photo proof -> instant replacement or
refund -> open AliExpress dispute to recover cost

Buyer remorse, high-value: return to UK address
(operations address), refund on receipt

Item never arrived: check tracking; if stalled/lost ->
refund customer immediately, dispute with supplier

------------------------------------------------------------

# DISPUTE RULES

Open AliExpress disputes within the protection window —
never let it lapse

Keep photo/video evidence from customers

Track recovery rate: refunds recovered from suppliers
