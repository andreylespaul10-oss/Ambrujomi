# PURPOSE

Define the evolution from generic dropshipping to a
defensible premium brand.

------------------------------------------------------------

# WHY PRIVATE LABEL

Higher margins (no middle listings)

Brand defensibility (cannot be price-compared 1:1)

Better quality control and packaging

Required for serious retail/press presence

------------------------------------------------------------

# ROADMAP

1. Prove winners via dropshipping data (bestsellers, reviews)

2. Negotiate custom packaging with Level 3 supplier
(logo box, thank-you card, care guide)

3. Small MOQ private run (200-500 units) to UK 3PL

4. Redesign listing with owned photography and branding

5. Retire the generic version; raise the price point

------------------------------------------------------------

# CANDIDATE CRITERIA

Stable demand 3+ months | Review rating 4.5+ |
Light/durable for storage | Margin supports MOQ investment
