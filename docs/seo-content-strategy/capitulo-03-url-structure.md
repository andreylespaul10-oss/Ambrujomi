# CHAPTER 3
# URL STRUCTURE
# ============================================================

# PURPOSE

Provide consistent, readable and SEO-friendly URLs.

------------------------------------------------------------

# URL PRINCIPLES

Short

Descriptive

Human-readable

Keyword-focused

Lowercase

Hyphen-separated

------------------------------------------------------------

# EXAMPLES

/

Homepage

/products

Category Index

/products/skincare

Category

/products/skincare/hydrating-serum

Product

/blog

Blog Homepage

/blog/how-to-build-a-skincare-routine

Article

/account

Customer Dashboard

/checkout

Checkout

------------------------------------------------------------

# URL RULES

No spaces

No special characters

No unnecessary parameters

Canonical URL required

------------------------------------------------------------

# REDIRECTS

301

Permanent

302

Temporary

Maintain SEO equity during migrations.
