# CHAPTER 4
# METADATA STANDARDS
# ============================================================

# PURPOSE

Define consistent metadata standards that improve search engine
visibility, click-through rates (CTR) and social media sharing.

Requirements

Compelling

Call to Action

140–160 Characters

Supported Values

index

noindex

follow

noarchive

nosnippet

index, follow
