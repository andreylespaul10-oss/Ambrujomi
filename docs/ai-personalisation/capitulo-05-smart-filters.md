# PURPOSE

Help customers narrow the catalogue quickly with filters that
adapt to context and behaviour.

------------------------------------------------------------

# FILTER TYPES

Standard: category, price range, rating, availability, offers

Beauty-specific: skin concern (dryness, dullness, puffiness),
skin type (dry, oily, combination, sensitive),
ingredient (vitamin C, collagen, hyaluronic acid),
material (jade, silk, silicone)

------------------------------------------------------------

# ADAPTIVE BEHAVIOUR

Filters reorder by usage frequency per category.

Irrelevant filters hide automatically (no "skin type" on diffusers).

Applied filters persist across the session.

Personalised default sort for returning customers.

------------------------------------------------------------

# BUSINESS RULES

Filter combinations must always be shareable via URL.

Filter counts must reflect live stock.

Mobile filters open in a bottom sheet, one-thumb reachable.
