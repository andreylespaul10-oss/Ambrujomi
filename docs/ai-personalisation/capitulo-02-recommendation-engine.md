# PURPOSE

Deliver personalised product recommendations that increase
conversion, basket size and repeat purchases.

------------------------------------------------------------

# RECOMMENDATION TYPES

Similar Products — same category, close price range

Frequently Bought Together — order history co-occurrence

Trending Products — sales velocity over the last 7 days

Recently Viewed — session and account history

Cross-Sell — complementary categories (serum -> LED mask)

Upsell — premium alternative within the same need

------------------------------------------------------------

# PLACEMENTS

Home page ("Recommended for you")

Product page ("Complete the ritual")

Basket ("You may also like")

Post-purchase email

Abandoned basket email

------------------------------------------------------------

# DATA INPUTS

View events, add-to-basket events, purchases,
wishlist items, search queries, category affinity,
skin-concern quiz answers (future).

------------------------------------------------------------

# BUSINESS RULES

Never recommend out-of-stock products.

Never recommend the product already in view or in basket.

Respect category exclusions (e.g. gift cards).

Fall back to best sellers when history is insufficient (cold start).

------------------------------------------------------------

# KPIS

Recommendation click-through rate

Attach rate (recommended items added to basket)

Revenue attributed to recommendations
