# PURPOSE

Understand how customers actually use the shop, to improve
UX and merchandising with evidence.

------------------------------------------------------------

# CAPABILITIES

Journey Analysis — common paths to purchase and drop-off

Clickstream Analysis — what customers see and skip

Heatmaps — attention on key templates (home, product, checkout)

Behaviour Clustering — browsing personas (researcher,
bargain hunter, gift buyer, loyal replenisher)

------------------------------------------------------------

# OUTPUTS

Weekly UX insight report in the admin dashboard

Drop-off alerts when checkout abandonment rises abnormally

Persona-based merchandising suggestions

------------------------------------------------------------

# PRIVACY RULES

Analytics run on consented data only (UK GDPR + PECR).

Session recordings mask all personal data fields.

Insights are aggregate; individual profiles are never exposed.
