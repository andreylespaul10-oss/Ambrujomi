# CHAPTER 1
# SECURITY ARCHITECTURE

## PURPOSE

The Security Architecture establishes the foundation for protecting
Bliss Glow against cyber threats while ensuring confidentiality,
integrity, availability and regulatory compliance.

Security must be integrated into every layer of the platform rather than
added as an afterthought.

------------------------------------------------------------

# SECURITY OBJECTIVES

The platform shall provide:

• Confidentiality

• Integrity

• Availability

• Accountability

• Non-repudiation

• Privacy by Design

• Security by Design

------------------------------------------------------------

# SECURITY PRINCIPLES

Every system must follow:

Least Privilege
