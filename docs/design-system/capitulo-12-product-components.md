# PURPOSE

Product Components define how every product is presented throughout the
Bliss Glow platform.

The objective is to maximise customer confidence, improve conversion
rates and create a premium shopping experience.

Every product component must remain reusable, responsive and fully
accessible.

------------------------------------------------------------

# DESIGN PRINCIPLES

Product presentation should be:

• Informative

• Elegant

• Consistent

• Trustworthy

• Fast

• Conversion-focused

# ACCESSIBILITY

Images require descriptive alternative text.

Buttons support keyboard navigation.

Reviews accessible via screen readers.

Colour is never the only indicator of stock status.

WCAG 2.2 AA compliant.

------------------------------------------------------------

# PERFORMANCE

Lazy-load images.

Optimise media.

Load reviews asynchronously.

Cache product recommendations.

Avoid layout shifts.
