# PURPOSE

Typography is one of the strongest elements of the Bliss Glow visual
identity. It should communicate elegance, readability and professionalism
while maintaining excellent accessibility across all devices.

The typography system must remain consistent throughout every digital
experience.

------------------------------------------------------------

# LINE LENGTH

Recommended

60–75 characters per line

Maximum

90 characters

Long paragraphs should be avoided.

------------------------------------------------------------

# TEXT ALIGNMENT

Default

Left aligned

Centre alignment

Only for hero sections, banners and selected marketing content.

Justified text is not permitted.

------------------------------------------------------------

# ACCESSIBILITY

Minimum body text

16 px

Minimum contrast ratio

4.5:1

Avoid light grey text on white backgrounds.

Maintain adequate spacing between headings and paragraphs.

------------------------------------------------------------

# RESPONSIVE TYPOGRAPHY

Desktop

Full scale

Tablet

Reduce headings by approximately 10%

Mobile

Reduce headings by approximately 20%

Body text remains at 16 px whenever possible.

------------------------------------------------------------

# BEST PRACTICES

Use consistent heading hierarchy.

Avoid excessive font weights.

Do not mix multiple font families within the same component.

Maintain sufficient whitespace around text.

Always prioritise readability over decoration.
