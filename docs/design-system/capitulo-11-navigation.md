# NAVIGATION
# Bliss Glow Design System
# Version 1.0
# ============================================================

# PURPOSE

The navigation system enables customers and administrators to move
through the Bliss Glow platform quickly, confidently and intuitively.

Navigation should minimise friction while making every important area
easy to discover.

------------------------------------------------------------

# DESIGN PRINCIPLES

Navigation should be:

• Clear

• Predictable

• Responsive

• Accessible

• Fast

• Consistent
