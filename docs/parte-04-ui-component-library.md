# COMPONENT PHILOSOPHY

Every UI element must be reusable.

Avoid duplicated code.

Every component must have:

- Default State
- Hover State
- Focus State
- Active State
- Disabled State
- Loading State
- Error State (when applicable)

All components must be responsive.

---

# Header Component

The header is visible on every page.

It must contain:

- Bliss Glow Logo
- Main Navigation
- Mega Menu
- Search Bar
- Wishlist Icon
- Shopping Cart
- User Account
- Language Selector
- Currency Selector
- Announcement Bar
- Sticky Navigation

Behavior:

- Transparent over hero sections.
- Solid background after scrolling.
- Smooth transition while scrolling.
- Mobile hamburger menu below tablet width.

---

# Announcement Bar

Position:

Top of the website.

- Free Shipping

- Holiday Campaigns

Must be dismissible.

# Animation Guidelines

Animation Duration:

150–300ms

Animation Types:

Fade

Slide

Scale

Opacity

Transform

Avoid excessive motion.

Animations should improve usability, never distract the user.
